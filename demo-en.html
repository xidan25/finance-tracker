finance-tracker · Demo

Once it's set up, you get three things: a local sync tool that runs daily, a Notion table you can browse anytime, and a monthly spending review.

Everything below is fictional demo data — no real transactions

1Local sync tool

A command-line program that runs on your own computer. Each day it reads email, classifies with AI, writes to a database, syncs to Notion, and pops a notification when done.
